Homebrew Patches
================

This repo is an organized location to store patches for formula in [homebrew]. 

If you want to contribute to this, please keep your submissions organized in the form:

    <formula_name>/<version>/<patch>.patch

you can put more subdirectories for arganization as long as the `<formula_name>/<version>/` part is preserved.


[homebrew]: <http://github.com/mxcl/homebrew> "The hombrew project"
